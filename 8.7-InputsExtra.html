INPUTS EXTRA

INPUT TIPUS COLOR

U input de tipus color ens permet priar un color en una paleta de colors. Si feu una pàgina on l'usuari hagi d'escolllir un color per algun item en concret o el color de fons d'una pagina us anira molt bé.

INPUT TIPUS NUMBER

Un input de tipus number només ens deixa posar números. Les petites fletxes que surten a la dreta per pujar o baixar el numero es diuen widgets. Aquest input s'ha d'utilitzar per temes numèrics que no siguin dates. Per demanar les dates ja en tenim d'especifics.

INPUT TIPUS RANGE

Un input de tipus range ens pinta una barra per seleccionar un valor. Accepta l'atribut step que determina de quant en quant es mou la nostra barra. També té l'atribut min i max.

Accepta l'atrbut stepque determina de quant en qant es pot mourela barra.

També accepta els atributs min i max

INPUT TIPUS RESET

Un input de tipus reset crea un botó que permet borrar tots els camps del formulari.
