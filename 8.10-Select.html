SELECT

Element html select basic

L'etiqueta select serveix per seleccionar entre una llista d'opcions un o més items

Si es posa l'etiqueta select sola, activa una llista desplegable peró no es veu res.

Etiqueta option

Cada etiqueta option afegeix un item per poder triar en el menú desplegable. El select porta l'atribut name i les enquestes option els atributs value.

L'atribut multiple que es posa dins l'etiqueta select, ens permet escollir-ne més d'un si apretem la tecla control.
